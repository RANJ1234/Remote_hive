Setup Instructions
How to set up Google Analytics
  1. Go to Google Analytics
  2. Create a new Analytics account (if needed)
  3. Create a new property for your website
  4. Complete the setup to get your Measurement ID
  5. Copy the ID and paste it in the form on the left
  6. Click "Save Tracking ID"
Important Notes
  • For Google Analytics 4, the ID starts with "G-"
  • For Universal Analytics, the ID starts with "UA-"
  • It can take up to 24 hours for data to start appearing in your Google Analytics account
  • Make sure you comply with privacy laws (GDPR, CCPA) when using analytics
